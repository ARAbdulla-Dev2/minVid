Implementation

Basic Usage

Add minVid to your project with this simple HTML structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Example minVid</title>    
</head>
<body>
    <video 
        id="video"
        class="absolute inset-0 w-full h-full object-contain"
        preload="metadata"
        poster="https://example.com/poster.jpg"
        height="640"
        width="360"
        vid-title="Your Video Title"
    >
        <source src="hd-video.mp4" type="video/mp4" quality="HD">
        <source src="sd-video.mp4" type="video/mp4" quality="SD" default>
        Your browser does not support the video tag.
    </video>
    
    <script src="./minVid1.0.0.js"></script>
</body>
</html>

Attributes Explained

  • vid-title - Sets the title displayed on the player
  • quality - Marks the quality of the video source (appears in quality selector)
  • default - Sets the default quality to load initially
