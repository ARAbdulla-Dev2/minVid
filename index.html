GET STARTED

Implementation Guide

Get minVid up and running in just a few simple steps

Basic Implementation

1. Include the script

<script src="https://cdn.jsdelivr.net/gh/ARAbdulla-Dev2/minVid/minVid1.0.0.js"></script>

2. Add your video element

<video 
    id="video"
    preload="metadata"
    poster="path/to/poster.jpg"
    height="360"
    width="640"
    vid-title="Your Video Title"
>
    <source src="hd-video.mp4" type="video/mp4" quality="HD">
    <source src="sd-video.mp4" type="video/mp4" quality="SD" default>
    Your browser does not support the video tag.
</video>

Pro Tip

The player will automatically initialize on all video elements in your page. No additional JavaScript required!

Configuration Options

Attribute Description Required
vid-title Sets the title displayed on the player Optional
quality Marks the quality of the video source (appears in quality selector) Required for multi-quality
default Sets the default quality to load initially Optional
